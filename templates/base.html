📌 자주 쓰는 기능
🖨️ 인쇄 — 의뢰서/성적서를 PDF로 출력
📥 엑셀 — 의뢰서/성적서/관리대장을 엑셀 다운로드
📒 관리대장 — 전체 시험 이력 한눈에 조회·필터
📈 통계 — 연/월별 적합률·커버리지율 차트
📋 시험 의뢰서 등록
① 왼쪽 메뉴 시험 의뢰서 → ＋ 클릭
② 의뢰 부서·담당자·제품 정보 입력
③ 4. 시험 항목에 항목 추가 (+ 행 추가)
④ 5. 판정 기준에 기준 내용 입력
⑤ 등록 후 자동으로 의뢰번호 발번 (RTN-YYYYMM-NNN)
🔍 목록 필터
상태 탭 — 접수대기/접수완료/시험중/결과회신/완료 클릭 필터
팀 필터 — 의뢰부서별 조회
검색 — 의뢰번호·제품명·부서명 검색
⚠️ 기한 초과 표시
완료희망일이 지난 경우 날짜 ⚠️ 표시
결과회신·완료·보류 상태는 기한 초과 제외
📊 결과서 등록
① 결과서 관리 목록 → 결과 입력 버튼
② 시험 항목별 결과(합격/불합격) 선택
③ 종합 판정: 적합 / 부적합 / 조건부적합
④ 저장 시 상태가 자동으로 결과회신으로 변경
🖨️ 성적서 출력
결과서 목록 → 인쇄 버튼 → 브라우저 인쇄 기능 사용
결과서 목록 → 엑셀 버튼 → .xlsx 다운로드
의뢰서 인쇄(🖨️)와 성적서 인쇄는 별도 버튼 사용
📌 판정 결과 의미
적합 — 모든 시험 항목 기준 충족
부적합 — 하나 이상의 항목 기준 미달
조건부적합 — 조건부로 기준 충족
⚠️ 부적합 등록
① 왼쪽 메뉴 부적합 등록 → ＋ 클릭
② 발생일·제품명·부적합 유형·심각도 입력
③ 관련 시험 의뢰서가 있으면 연결 선택
④ 자동으로 번호 발번 (NC-YYYYMM-NNN)
🔄 처리 단계
등록 → 원인분석중 → CAPA진행 → 개선완료 → 종료
개선조치 관리에서 단계별 조치 내용 기록
📊 심각도 기준
상 — 즉각 대응 필요, 출하 영향
중 — 일정 내 처리 필요
하 — 일반적 개선 사항
🏷️ 의뢰서 상태 설명
접수대기 — 의뢰서 등록 후 품질팀 확인 전
접수완료 — 품질팀이 의뢰서 접수 확인 완료
시험중 — 시험 진행중
결과회신 — 시험 결과 등록 완료, 의뢰자에게 통보
완료 — 모든 처리 완료
보류 — 시험 일시 중단
🔔 알림 기준
기한 초과 — 완료희망일 경과, 결과회신·완료·보류 제외
마감 임박 — 완료희망일 7일 이내, 대시보드에 표시
📒 시험 관리대장
전체 시험 이력을 표 형태로 조회
연도·월·부서·상태·키워드 필터 지원
시험 항목별 결과(합격/불합격) 함께 표시
엑셀 저장·인쇄 기능 지원
📅 시험 일정 캘린더
월간 캘린더로 시험 시작일·완료희망일 시각화
날짜 클릭 시 해당 의뢰서 목록 표시
📈 통계 / 모니터링
연도별·월별·주별 의뢰·완료 추이 차트
적합률·커버리지율 게이지 및 라인 차트
부서별·제품별 의뢰 건수 막대 차트
👥 회원 관리 (관리자 전용)
사용자 등록·수정·권한 관리
부서별 접근 권한: 품질팀만 접수·완료 처리 가능
