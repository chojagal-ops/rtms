===================================================
 신뢰성 시험 관리 시스템 (RTMS)
 Reliability Test Management System
===================================================

[ 실행 방법 ]
1. run.bat 파일을 더블클릭
2. 브라우저에서 http://localhost:5001 접속
3. 기본 관리자 계정: admin / admin1234

[ 주요 기능 ]
- 신뢰성 시험 의뢰서 등록 / 수정 / 조회
- 시험 항목 및 판정 기준 관리
- 신뢰성 시험 결과서 입력 및 관리
- 기존 엑셀 의뢰서/결과서 파일 업로드 자동 입력
- 상태별 현황 대시보드 (접수대기/접수완료/시험중/결과회신/완료)
- 회원 가입 및 관리자 승인

[ 의뢰서 번호 채번 규칙 ]
  RTN-YYYYMM-NNN
  예) RTN-202511-001

[ 파일 구조 ]
  app.py          - 메인 서버 실행 파일
  models.py       - 데이터베이스 구조 정의
  data.db         - 저장 데이터 (자동 생성)
  static/uploads/ - 첨부파일 저장 폴더
  .env            - 비밀 설정 (외부 공유 금지)

[ 포트 변경 ]
  .env 파일에서 FLASK_PORT=5001 수정
